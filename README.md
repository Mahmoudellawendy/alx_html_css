# CSS Advanced Project

This project is part of the **ALX Front-End curriculum**, focusing on **advanced CSS styling** and how to convert a Figma design into a real web page.

## 🧠 Objective

- Apply advanced CSS concepts to bring a Figma design to life.
- Refine an existing HTML file by integrating images, fonts, and styles.
- Understand how to read Figma specs and apply them to CSS.
